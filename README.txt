Campus Cafeteria Ordering System

Setup
1. Extract the folder into:
   C:\xampp\htdocs\campus_cafeteria

2. Start Apache and MySQL in XAMPP.

3. Open phpMyAdmin and import:
   database/campus_cafeteria.sql

4. Visit:
   http://localhost/campus_cafeteria/
